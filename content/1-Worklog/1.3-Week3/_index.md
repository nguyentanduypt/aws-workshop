### Week 3 Objectives:

- Explore and utilize the Amazon S3 object storage service.
- Gain advanced hands-on experience with Amazon EC2 and learn how to manage the lifecycle of cloud resources.

### Week 3 Achievements:

- Developed a solid understanding of **Amazon S3** and its storage capabilities:
  - Learned how to create and manage **S3 Buckets** and configure secure access permissions for stored objects.
  - Understood how Amazon S3 can be used to host and manage static assets such as images, videos, and documents for web applications.
- Strengthened practical skills with **Amazon EC2**:
  - Learned how to use **User Data** scripts to automate software installation and server configuration during instance launch.
  - Gained experience in managing the lifecycle of EC2 instances, including stopping unused resources to optimize cloud costs.
- Improved overall knowledge of AWS storage and compute services, establishing a strong technical foundation for developing subsequent modules of the E-commerce system.
